# KudoSpot

This project can be cloned locally and run using a single command
### `npm install`
### `npm start`

This will start the server and build the application available at `http://localhost:3000`

## Technologies Used

- **Backend**: Node.js, Express.js
- **Frontend**: ReactJS
- **Database**: MongoDB
